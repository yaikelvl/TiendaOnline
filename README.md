# Tienda Online

1. Clonar proyecto

2. Instalar los modulos de Node.
```
yarn install
```

3. Levantar la Base de Datos.
```
docker-compose up -d
```

4. Renombrar el fichero ```.env.template``` a ```.env```

5. Cambiar las variables de entorno.

6. Levantar modo de desarrollo.
```
yarn start:dev
```
